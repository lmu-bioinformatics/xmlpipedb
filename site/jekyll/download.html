Download

Regular GenMAPP users can go straight to downloading Gene Databases for their species of interest. Software developers in bioinformatics (or other areas) that wish to automate the creation of relational databases from XML sources will be interested in the releases available from SourceForge.

System Requirements

Recommended Hardware

  • For Gene Databases, GenMAPP, and exporting Gene Databases with GenMAPP Builder, a Microsoft Windows machine is required.
  • At least 1G of memory is recommended. The software may run with less memory, but its performance and stability will degrade.
  • For all other functions, XMLPipeDB software will run on any machine that can run Java and PostgreSQL.

Gene Databases

  • GenMAPP software

GenMAPP Builder

  • Java 6 Runtime Environment
  • Relational database with available JDBC driver (only PostgreSQL has been explicitly tested so far)
  • Microsoft Jet Database Engine (for exporting Gene Databases); included with Microsoft Windows

XSD-to-DB

  • Java 6 Runtime Environment

XMLPipeDB Utilities

  • Java 6 Runtime Environment
  • Relational database with available JDBC driver (only PostgreSQL has been explicitly tested so far)

Database Libraries (UniProtDB, GODB)

  • Java 6 Runtime Environment
  • Relational database with available JDBC driver (only PostgreSQL has been explicitly tested so far)
